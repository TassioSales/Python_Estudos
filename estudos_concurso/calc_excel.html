Guia de Planilhas

Excel & Calc para Concursos

Bem-vindo(a) ao Guia Definitivo de Planilhas!

Este material foi criado para ser sua principal ferramenta de estudo sobre Microsoft Excel e LibreOffice Calc, dois dos softwares mais cobrados em provas de informática para concursos públicos.

O domínio de planilhas eletrônicas não é apenas um requisito técnico, mas uma habilidade estratégica que permite analisar dados, otimizar tarefas e tomar decisões informadas – competências essenciais para qualquer servidor público. Em um governo cada vez mais digital, a capacidade de extrair inteligência de dados é um diferencial para a eficiência e transparência na gestão pública.

Nosso Objetivo

Fornecer um conteúdo claro, direto e focado no que realmente cai nas provas, com exemplos práticos, comparações e dicas para você gabaritar as questões sobre planilhas.

Como Navegar

Use o menu lateral para navegar pelos tópicos. Recomendamos seguir a ordem para um aprendizado progressivo, mas sinta-se à vontade para consultar os temas conforme sua necessidade.

Resumo Estratégico para Provas

As bancas examinadoras costumam focar em um conjunto específico de competências. Concentre seus estudos nos seguintes pontos-chave:

  • Referências de Célula: Entender a diferença entre relativas (A1), absolutas ($A$1) e mistas (A$1, $A1) é o tópico MAIS IMPORTANTE. A maioria das questões de fórmulas complexas depende desse conceito.
  • Funções Essenciais: Domine o uso de `SE`, `CONT.SE`, `SOMASES` e `PROCV`. Saiba a sintaxe correta, a ordem dos argumentos e a aplicação prática de cada uma.
  • Diferenças entre Excel e Calc: Conheça as pequenas, mas cruciais, diferenças de nomenclatura de funções e atalhos de teclado (ex: `Ctrl+N` vs `Ctrl+B` para negrito).
  • Ferramentas de Dados: Compreenda a finalidade e a aplicação de `Filtros`, `Classificação`, `Formatação Condicional` e `Validação de Dados`.

Dica de Estudo: Não se limite a decorar. Crie pequenas planilhas para testar cada função e recurso. A prática consolida o conhecimento teórico e ajuda a identificar as "pegadinhas" das provas.

Interface e Conceitos Fundamentais

Toda planilha é uma grade formada por linhas e colunas. Entender sua estrutura é o primeiro passo.

Coluna

Identificada por letras (A, B, C...). É o conjunto vertical de células.

Linha

Identificada por números (1, 2, 3...). É o conjunto horizontal de células.

Célula

A interseção de uma linha e uma coluna (ex: A1). É onde os dados são inseridos.

Referências de Célula: O Ponto-Chave

Este é o conceito mais crucial para dominar fórmulas. Uma referência indica à fórmula onde encontrar os valores a serem usados.

Atenção: O caractere `$` é o que "trava" (fixa) uma referência, tornando-a absoluta. Pressione a tecla F4 com o cursor sobre uma referência para alternar entre os tipos (A1 → $A$1 → A$1 → $A1).

1. Referência Relativa (ex: `A1`)

É o padrão. Ao copiar ou arrastar uma fórmula, ela se ajusta automaticamente à nova posição. Se você copia `=A1+B1` da célula C1 para C2, a fórmula se torna `=A2+B2`.

2. Referência Absoluta (ex: `$A$1`)

A referência permanece fixa, não importa para onde a fórmula seja copiada. O `$` trava tanto a coluna `A` quanto a linha `1`. Ideal para constantes, como uma taxa de juros ou um valor de comissão.

3. Referência Mista (ex: `$A1` ou `A$1`)

Trava apenas a coluna (`$A1`) ou apenas a linha (`A$1`). É útil em tabelas de multiplicação e cenários mais complexos.

Fórmulas e Funções Básicas

As funções básicas realizam os cálculos mais comuns. A sintaxe é simples: `=NOME_DA_FUNÇÃO(argumento1; argumento2; ...)`.

Pegadinha de Prova: No padrão brasileiro, o separador de argumentos em uma função é o **ponto e vírgula (`;`)**, não a vírgula (`,`). A vírgula é o separador decimal.

SOMA

Adiciona todos os números em um intervalo de células.

=SOMA(A1:A10) // Soma os valores de A1 até A10

MÉDIA

Retorna a média aritmética dos argumentos.

=MÉDIA(B1:B5) // Calcula a média de B1 a B5

MÁXIMO e MÍNIMO

Retornam o maior e o menor valor de um conjunto.

=MÁXIMO(C1:C100)
=MÍNIMO(C1:C100)

CONT.NÚM / CONT.VALORES

`CONT.NÚM` conta células com números. `CONT.VALORES` conta células não vazias.

=CONT.NÚM(D1:D10)
=CONT.VALORES(D1:D10)

HOJE e AGORA

`HOJE()` retorna a data atual. `AGORA()` retorna a data e a hora atuais. São funções voláteis, atualizadas sempre que a planilha é recalculada.

=HOJE()
=AGORA()

Funções Intermediárias e Avançadas

Estas funções permitem criar planilhas mais inteligentes e dinâmicas, sendo muito exploradas em provas.

SE

Realiza um teste lógico e retorna um valor se o teste for verdadeiro, e outro se for falso. Essencial para tomada de decisões.

Sintaxe: `=SE(teste_lógico; valor_se_verdadeiro; valor_se_falso)`

=SE(A1>10; "Aprovado"; "Reprovado")

CONT.SE e SOMASES

`CONT.SE` conta células que atendem a um critério. `SOMASES` soma células que atendem a múltiplos critérios.

Sintaxe CONT.SE: `=CONT.SE(intervalo; critérios)`

Sintaxe SOMASES: `=SOMASES(intervalo_soma; intervalo_critérios1; critérios1; ...)`

=CONT.SE(B1:B50; "Marketing")
=SOMASES(C1:C50; B1:B50; "Vendas"; D1:D50; ">1000")

PROCV (VLOOKUP)

Procura um valor na primeira coluna de uma tabela e retorna um valor na mesma linha de uma coluna que você especificar.

Sintaxe: `=PROCV(valor_procurado; matriz_tabela; núm_índice_coluna; [procurar_intervalo])`

Pegadinha Clássica: O PROCV só procura na **primeira coluna** da matriz e só retorna valores à **direita**. O último argumento, `[procurar_intervalo]`, é crucial: `FALSO` para correspondência exata (mais comum) e `VERDADEIRO` para aproximada.

=PROCV(A1; C1:E10; 3; FALSO)

ÍNDICE e CORRESP: A Dupla Dinâmica

Esta combinação supera as limitações do PROCV. `CORRESP` encontra a posição de um item, e `ÍNDICE` retorna o valor nessa posição.

Sintaxe: `=ÍNDICE(matriz_retorno; CORRESP(valor_procurado; matriz_pesquisa; 0))`

Dica de Mestre: Use ÍNDICE/CORRESP sempre que precisar de uma busca flexível. Diferente do PROCV, esta combinação permite que a coluna de retorno esteja à esquerda da coluna de pesquisa e não quebra se novas colunas forem inseridas na tabela.

=ÍNDICE(A1:A10; CORRESP("ProdutoX"; B1:B10; 0)) // Procura "ProdutoX" na coluna B e retorna o valor correspondente da coluna A.

SEERRO

Trata erros em fórmulas, retornando um valor alternativo caso a fórmula original resulte em erro (ex: #N/D, #DIV/0!).

Sintaxe: `=SEERRO(valor; valor_se_erro)`

Dica: Use `SEERRO` para tornar suas planilhas mais limpas e profissionais, evitando a exibição de mensagens de erro para o usuário final.

=SEERRO(A1/B1; "Cálculo Inválido")

Formatação, Filtros e Validação

Recursos que organizam, destacam e protegem a integridade dos seus dados.

Formatação Condicional

Altera a aparência das células com base em regras. Ótimo para destacar informações importantes.

Exemplos: Colorir valores acima da média, adicionar barras de dados para visualização rápida, realçar os 10 maiores valores.

Onde encontrar: Excel: Guia `Página Inicial`. Calc: Menu `Formatar`.

Validação de Dados

Restringe o tipo de dados ou os valores que os usuários podem inserir em uma célula.

Exemplo: Criar uma lista suspensa de opções (ex: "Pendente", "Em Andamento", "Concluído") para uma célula de status.

Onde encontrar: Excel: Guia `Dados`. Calc: Menu `Dados > Validade`.

Remover Duplicadas

Ferramenta que examina um intervalo de dados e exclui linhas que são inteiramente duplicadas ou com base em colunas específicas que você selecionar.

Onde encontrar: Excel: Guia `Dados`. Calc: `Dados > Filtro > Filtro Padrão` (com condição `<>`).

Texto para Colunas

Divide o conteúdo de uma coluna de texto em várias colunas, com base em um delimitador (como vírgula, ponto e vírgula ou espaço) ou em larguras de campo fixas.

Onde encontrar: Guia `Dados` em ambos os programas.

Formatar como Tabela (Excel) / Estilos Automáticos (Calc)

Transforma um intervalo de dados em uma tabela estruturada, que oferece benefícios como filtros automáticos, linhas em cores alternadas para melhor leitura e expansão automática de fórmulas e formatação quando novas linhas são adicionadas.

Dica de Mestre: Sempre use "Formatar como Tabela" para seus conjuntos de dados. Isso torna a criação de gráficos e tabelas dinâmicas muito mais robusta, pois eles se referem ao nome da tabela e se atualizam automaticamente quando novos dados são inseridos.

Gráficos e Tabelas Dinâmicas

Ferramentas poderosas para visualizar e resumir dados.

Tipos de Gráficos Essenciais

Colunas/Barras: Ideal para comparar valores entre diferentes categorias.
Linhas: Perfeito para mostrar tendências e a evolução de uma variável ao longo do tempo.
Pizza: Mostra a proporção de cada parte em relação a um todo.

Tabela e Gráfico Dinâmicos

A **Tabela Dinâmica** resume dados interativamente. O **Gráfico Dinâmico** é uma representação visual da Tabela Dinâmica, que se atualiza automaticamente com os filtros aplicados.

Dica de Prova: Entenda o conceito! Uma Tabela Dinâmica serve para **resumir, agrupar e analisar** dados. As bancas adoram questões conceituais sobre sua finalidade.

Dicas e Pegadinhas de Prova

Fique atento a estes detalhes que podem fazer a diferença entre o acerto e o erro.

1. Operador `&`

A função `CONCATENAR` une textos, mas o operador `&` faz a mesma coisa. As bancas podem colocar as duas opções para confundir.

2. Datas são Números

O Excel/Calc armazena datas como números sequenciais. Isso permite fazer cálculos matemáticos com elas (ex: `A1+10`).

3. `CONT.SE` vs `CONT.SES`

`CONT.SE` avalia um critério. `CONT.SES` (com 'S' no final) avalia múltiplos critérios. O mesmo vale para `SOMA.SE` e `SOMASES`.

4. Apagar vs. Limpar

Pressionar `Delete` apaga o conteúdo, mas mantém a formatação. Para remover tudo, use a opção "Limpar Tudo".

5. `PROCV` e o `FALSO`

O último argumento do `PROCV` é opcional, mas crucial. Omiti-lo ou usar `VERDADEIRO` busca uma correspondência aproximada, o que quase sempre leva a erros se não for intencional. Para buscas exatas, sempre use `FALSO` ou `0`.

6. `SOMA` ignora texto

A função `SOMA` ignora células que contêm texto. Se você tentar somar um intervalo que inclui textos, ela simplesmente não os considerará no cálculo, sem gerar erro.

7. `MÉDIA` também ignora texto

Assim como a `SOMA`, a `MÉDIA` ignora células com texto. Ela calcula a média apenas dos valores numéricos no intervalo, dividindo pela contagem de números, não pelo total de células.

8. Referência Circular

Ocorre quando uma fórmula se refere, direta ou indiretamente, à sua própria célula (ex: fórmula em A1 contém `...A1...`). Isso geralmente é um erro e o programa exibirá um aviso.

9. Ocultar vs. Excluir

Ocultar uma linha ou coluna a remove da exibição, mas seus dados ainda existem e são considerados em cálculos de fórmulas que abrangem aquele intervalo. Excluir remove os dados permanentemente.

10. Precedência de Operadores

Lembre-se da ordem: Parênteses, Exponenciação, Multiplicação/Divisão, Adição/Subtração. Uma questão pode testar se `2+3*5` é 25 (errado) ou 17 (certo).

Exercícios Resolvidos de Concursos

(CEBRASPE/CESPE - 2023)

Considere a planilha a seguir no Microsoft Excel. Se na célula C2 for inserida a fórmula `=SE(B2>3; "Caro"; "Barato")`, qual será o resultado exibido?

AB
1ProdutoValor
2Caneta5
3Lápis3

Resposta Correta: "Caro"

Comentário: A função `SE` testa a condição `B2>3`. O valor em B2 é 5, que é maior que 3. Portanto, a condição é verdadeira, e a função retorna o primeiro resultado, "Caro".

(FCC - 2022)

Em uma planilha do LibreOffice Calc, para fixar a referência à célula A1 de modo que ela não se altere ao copiar a fórmula, deve-se usar:

Resposta Correta: `$A$1`

Comentário: O caractere `$` é usado para criar uma referência absoluta, travando a coluna (A) e a linha (1). Esta é uma das questões mais recorrentes sobre planilhas em concursos.

(FGV - 2023)

Dada a fórmula `=CONT.SE(A1:A5; ">10")` aplicada a um intervalo com os valores 5, 10, 15, 20, 25. Qual será o resultado?

Resposta Correta: 3

Comentário: A função conta quantas células no intervalo A1:A5 contêm um valor estritamente maior que 10. Os valores que atendem a esse critério são 15, 20 e 25, totalizando 3 células.

(VUNESP - 2022)

A fórmula `=MÉDIA(A1:A4)` é inserida em uma planilha onde A1=10, A2=20, A3="texto", A4=30. Qual o resultado?

Resposta Correta: 20

Comentário: A função `MÉDIA` ignora células que contêm texto. Portanto, o cálculo será a soma dos números (10+20+30=60) dividida pela quantidade de números (3), resultando em 20.

(AOCP - 2023)

Qual a finalidade principal do recurso Tabela Dinâmica no Microsoft Excel?

Resposta Correta: Resumir, analisar e apresentar grandes volumes de dados de forma interativa.

Comentário: Questões conceituais são comuns. A essência da Tabela Dinâmica não é criar gráficos ou formatar, mas sim resumir e analisar dados complexos sem o uso de fórmulas manuais.

(IBFC - 2022)

Na célula C1, a fórmula é `=$A1+B$1`. Se esta fórmula for copiada para a célula D2, qual será a fórmula resultante?

Resposta Correta: `=$A2+C$1`

Comentário: Esta questão testa referências mistas. A coluna A está travada (`$A`), então permanece A. A linha 1 é relativa, e ao mover uma linha para baixo (de 1 para 2), ela se torna 2. A coluna B é relativa, e ao mover uma coluna para a direita (de B para C), ela se torna C. A linha 1 está travada (`$1`), então permanece 1.

(CONSULPLAN - 2023)

Para somar os valores da coluna C apenas quando a coluna A for "Sul" e a coluna B for "Online", qual função deve ser utilizada?

Resposta Correta: `SOMASES`

Comentário: Como existem múltiplos critérios (Região="Sul" E Canal="Online"), a função correta é a `SOMASES`, que permite a soma baseada em diversas condições simultâneas. `SOMA.SE` só permitiria um critério.

(CEBRASPE/CESPE - 2022)

No LibreOffice Calc, o atalho de teclado para aplicar o formato de Negrito a uma célula selecionada é:

Resposta Correta: `Ctrl + B`

Comentário: Uma clássica questão de diferença entre os programas. Enquanto no Excel brasileiro o atalho é `Ctrl + N`, no Calc é `Ctrl + B` (de "Bold", em inglês), um padrão mais universal.

Glossário Técnico Simplificado

Argumento

Os valores que uma função utiliza para realizar operações. São separados por ponto e vírgula (;).

Concatenar

Unir duas ou mais cadeias de texto em uma só.

Dashboard

Um painel visual que resume informações e métricas importantes, geralmente usando gráficos e tabelas.

Intervalo (Range)

Um conjunto de células selecionadas. Ex: `A1:B10`.

Macro

Uma sequência de comandos e ações que pode ser gravada e executada para automatizar tarefas repetitivas.

Sintaxe

As regras de escrita de uma fórmula ou função para que o programa a entenda corretamente.
