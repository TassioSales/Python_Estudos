Bem-vindo(a) ao Guia Definitivo de Planilhas!

Este guia interativo é sua ferramenta completa para dominar o Microsoft Excel e o LibreOffice Calc, os dois softwares de planilhas mais cobrados em concursos públicos. Nosso Objetivo

Capacitar você com um material didático e estratégico que combina teoria, prática e dicas específicas para concursos. Cada seção foi projetada para reforçar sua compreensão e prepará-lo para os desafios das provas.

Dica: Pratique com planilhas reais enquanto estuda. Crie tabelas e teste as fórmulas para internalizar os conceitos.

Dica de Mestre: Foque nas seções de "Dicas de Prova" e "Exercícios" para entender o estilo das questões e evitar pegadinhas.

Por que Planilhas?

Planilhas são ferramentas-chave em concursos porque testam raciocínio lógico, atenção aos detalhes e habilidades práticas. Dominá-las demonstra sua capacidade de gerenciar dados, uma competência essencial no serviço público moderno.

Resumo Estratégico para Provas

Este resumo destaca os tópicos mais cobrados em concursos públicos, com foco em Excel e Calc. Priorize esses conceitos para maximizar seus acertos.

  • Referências de Célula: Entenda profundamente as referências relativas (A1), absolutas ($A$1) e mistas (A$1, $A1). Elas são a base para fórmulas dinâmicas e aparecem em 80% das questões de planilhas. Exemplo: Copiar `=A1*2` de C1 para C2 altera para `=A2*2`, mas `=$A$1*2` permanece fixo.
  • Funções Essenciais: Domine `SE`, `CONT.SE`, `SOMASES`, `PROCV` e `ÍNDICE/CORRESP`. Saiba a sintaxe exata e pratique com cenários reais. Exemplo: `=SE(A1>=60; "Aprovado"; "Reprovado")` testa uma condição e retorna um resultado.
  • Ferramentas de Dados: Aprenda a usar `Filtros`, `Classificação`, `Formatação Condicional` e `Validação de Dados`. Essas ferramentas aparecem em questões práticas e conceituais. Exemplo: A Formatação Condicional pode destacar automaticamente células com valores acima de 100.
  • Atalhos e Diferenças: Memorize atalhos como `Ctrl + N` (Excel) vs. `Ctrl + B` (Calc) para negrito. Bancas adoram testar essas diferenças. Veja a seção "Excel x Calc" para detalhes.

Dica de Estudo: Crie uma planilha com exemplos para cada tópico. Teste fórmulas como `=CONT.SE(A1:A10; ">100")` ou `=PROCV("João"; A1:C10; 3; FALSO)` com dados reais para fixar o aprendizado.

Estratégia de Prova: Leia atentamente o enunciado e identifique se a questão é sobre Excel ou Calc. Pequenas diferenças, como o atalho para negrito ou o separador de argumentos (`;` vs. `,`), podem ser decisivas.

Interface e Conceitos Fundamentais

Planilhas são grades formadas por linhas e colunas, organizadas em células. Compreender sua estrutura é essencial para dominar Excel e Calc.

Coluna

Identificada por letras (A, B, C...). Representa uma série vertical de células. Exemplo: A coluna B contém todas as células de B1 a B1048576 no Excel.

Linha

Identificada por números (1, 2, 3...). Representa uma série horizontal de células. Exemplo: A linha 5 inclui células como A5, B5, etc.

Célula

Interseção de uma linha e uma coluna (ex: A1). É a unidade básica para inserir dados, fórmulas ou funções.

Referências de Célula: O Coração das Planilhas

Referências de célula determinam como as fórmulas acessam dados. Este é o conceito mais testado em concursos, pois afeta diretamente o comportamento das fórmulas.

Atenção: Use a tecla F4 para alternar rapidamente entre tipos de referência (A1 → $A$1 → A$1 → $A1). Isso economiza tempo e reduz erros.

1. Referência Relativa (ex: `A1`)

Adapta-se à nova posição quando copiada. Exemplo: Se `=A1*2` está em C1 e for copiada para C2, torna-se `=A2*2`.

=A1+B1 // Copiado de C1 para D2, vira =B2+C2

2. Referência Absoluta (ex: `$A$1`)

Permanece fixa, independentemente de onde a fórmula é copiada. Ideal para constantes, como taxas ou valores fixos.

=A1*$A$1 // $A$1 permanece fixo ao copiar

Exemplo Prático: Para calcular o imposto de várias vendas usando uma taxa fixa em A1, use `=B2*$A$1` para garantir que A1 não mude ao arrastar a fórmula.

3. Referência Mista (ex: `$A1` ou `A$1`)

Trava apenas a coluna (`$A1`) ou a linha (`A$1`). Útil em tabelas complexas, como matrizes de multiplicação.

=B1*$A1 // Coluna A fixa, linha ajusta

Exemplo Avançado: Em uma tabela de multiplicação, use `=$A2*B$1` para criar uma grade onde a primeira linha e coluna são fixas, mas os valores internos se ajustam dinamicamente.

Fórmulas e Funções Básicas

As funções básicas são a porta de entrada para cálculos em planilhas. Elas seguem a sintaxe `=NOME_DA_FUNÇÃO(argumento1; argumento2; ...)`. Pratique cada uma com exemplos reais.

Pegadinha de Prova: No padrão brasileiro, use **ponto e vírgula (`;`)** como separador de argumentos e **vírgula (`,`)** como separador decimal. Exemplo: `=SOMA(1,5;2,5)`.

SOMA

Soma os valores numéricos em um intervalo, ignorando texto.

=SOMA(A1:A10) // Soma de A1 até A10

Exemplo Prático: Use `=SOMA(A1:A10)` para calcular o total de vendas em uma coluna de valores. Teste com valores mistos (números e texto) para confirmar que o texto é ignorado.

MÉDIA

Calcula a média aritmética de números, ignorando texto.

=MÉDIA(B1:B5) // Média de B1 a B5

Exemplo Prático: Aplique `=MÉDIA(A1:A5)` em uma coluna com notas (ex: 8, 7, "Ausente", 9, 6). O resultado será (8+7+9+6)/4 = 7,5.

MÁXIMO e MÍNIMO

Retornam o maior e o menor valor de um intervalo.

=MÁXIMO(C1:C100)
=MÍNIMO(C1:C100)

Exemplo Prático: Use `=MÁXIMO(B1:B10)` para encontrar o maior faturamento mensal em uma lista de vendas.

CONT.NÚM / CONT.VALORES

`CONT.NÚM` conta células com números ou datas. `CONT.VALORES` conta células não vazias, incluindo texto.

=CONT.NÚM(D1:D10)
=CONT.VALORES(D1:D10)

Dica de Mestre: Use `CONT.VALORES` para verificar quantas células foram preenchidas em um formulário. Use `CONT.NÚM` para contar apenas respostas numéricas.

HOJE e AGORA

`HOJE()` retorna a data atual. `AGORA()` retorna data e hora atuais. Ambas são voláteis (atualizam ao recalcular).

=HOJE()
=AGORA()

Pegadinha: Essas funções dependem da data/hora do sistema. Em provas, preste atenção ao formato de saída (ex: DD/MM/AAAA).

Funções Intermediárias e Avançadas

Essas funções permitem análises mais complexas e são frequentemente testadas em questões de nível intermediário a avançado.

SE

Executa um teste lógico e retorna valores diferentes para verdadeiro ou falso.

Sintaxe: `=SE(teste_lógico; valor_se_verdadeiro; valor_se_falso)`

=SE(A1>10; "Aprovado"; "Reprovado")

Exemplo Prático: Em uma planilha de notas, use `=SE(B2>=7; "Aprovado"; "Reprovado")` para classificar alunos automaticamente.

CONT.SE e SOMASES

`CONT.SE` conta células que atendem a um critério. `SOMASES` soma valores com base em múltiplos critérios.

Sintaxe CONT.SE: `=CONT.SE(intervalo; critério)`

Sintaxe SOMASES: `=SOMASES(intervalo_soma; intervalo_critério1; critério1; ...)`

=CONT.SE(B1:B50; "Marketing")
=SOMASES(C1:C50; B1:B50; "Vendas"; D1:D50; ">1000")

Exemplo Avançado: Para somar salários de funcionários de "Vendas" com mais de 5 anos de empresa, use `=SOMASES(C1:C100; B1:B100; "Vendas"; D1:D100; ">5")`.

PROCV (VLOOKUP)

Busca um valor na primeira coluna de uma tabela e retorna um valor da mesma linha em outra coluna.

Sintaxe: `=PROCV(valor_procurado; matriz_tabela; núm_índice_coluna; [procurar_intervalo])`

=PROCV(A1; C1:E10; 3; FALSO)

Pegadinha Clássica: O `PROCV` só busca na primeira coluna e só retorna valores à direita. Use `FALSO` para correspondência exata.

Exemplo Prático: Para encontrar o preço de um produto em A1, use `=PROCV(A1; B1:D100; 3; FALSO)` onde B1:D100 contém código, nome e preço.

ÍNDICE e CORRESP

`ÍNDICE` retorna um valor em uma posição específica. `CORRESP` encontra a posição de um valor. Juntas, superam limitações do `PROCV`.

Sintaxe: `=ÍNDICE(matriz_retorno; CORRESP(valor_procurado; matriz_pesquisa; 0))`

=ÍNDICE(A1:A10; CORRESP("ProdutoX"; B1:B10; 0))

Dica de Mestre: Use `ÍNDICE/CORRESP` para buscas à esquerda do valor procurado ou em tabelas dinâmicas onde colunas podem ser inseridas.

Exemplo Prático: Para retornar o nome de um produto cujo código está em B1, use `=ÍNDICE(A1:A100; CORRESP(B1; C1:C100; 0))`.

SEERRO

Evita mensagens de erro, retornando um valor alternativo se a fórmula falhar.

Sintaxe: `=SEERRO(valor; valor_se_erro)`

=SEERRO(A1/B1; "Cálculo Inválido")

Exemplo Prático: Use `=SEERRO(PROCV(A1; B1:D100; 3; FALSO); "Não encontrado")` para evitar `#N/D` em buscas malsucedidas.

CONCATENAR e &

Unem textos de várias células ou valores. O operador `&` é uma alternativa à função `CONCATENAR`.

Sintaxe: `=CONCATENAR(texto1; texto2; ...)` ou `=texto1 & texto2`

=CONCATENAR(A1; " "; B1)
=A1 & " " & B1

Exemplo Prático: Para criar um nome completo, use `=CONCATENAR(A1; " "; B1)` onde A1 é "João" e B1 é "Silva", resultando em "João Silva".

Formatação, Filtros e Validação

Essas ferramentas organizam dados, melhoram a visualização e garantem consistência nas entradas.

Formatação Condicional

Aplica formatações (cores, ícones, barras) com base em regras.

Exemplos: Destacar valores acima de 100, usar ícones para indicar desempenho ou barras de dados para comparar valores.

Onde encontrar: Excel: Guia `Página Inicial`. Calc: Menu `Formatar`.

Exemplo Prático: Aplique formatação condicional para colorir células com valores negativos de vermelho em uma coluna de lucros.

Validação de Dados

Restringe entradas em células para garantir consistência (ex: listas suspensas, números inteiros).

Exemplo: Criar uma lista suspensa com "Sim", "Não" ou "Talvez".

Onde encontrar: Excel: Guia `Dados`. Calc: Menu `Dados > Validade`.

Dica de Mestre: Use validação para evitar erros em formulários, como limitar uma célula a datas entre 01/01/2023 e 31/12/2023.

Remover Duplicadas

Elimina linhas duplicadas com base em colunas selecionadas.

Onde encontrar: Excel: Guia `Dados`. Calc: `Dados > Filtro > Filtro Padrão`.

Exemplo Prático: Remova duplicatas em uma lista de e-mails para garantir que cada endereço apareça apenas uma vez.

Texto para Colunas

Divide texto em várias colunas com base em um delimitador (ex: vírgula, espaço).

Onde encontrar: Guia `Dados` em ambos os programas.

Exemplo Prático: Converta "João,Silva,30" em três colunas: Nome, Sobrenome e Idade, usando a vírgula como delimitador.

Formatar como Tabela (Excel) / Estilos Automáticos (Calc)

Transforma dados em tabelas estruturadas com filtros, formatação e expansão automática.

Dica de Mestre: Tabelas estruturadas facilitam a criação de gráficos dinâmicos e tabelas dinâmicas, pois referenciam nomes de tabelas que se atualizam automaticamente.

Exemplo Prático: Converta um intervalo de vendas em uma tabela para aplicar filtros automáticos e calcular totais dinâmicos.

Gráficos e Tabelas Dinâmicas

Essas ferramentas transformam dados brutos em visualizações e resumos poderosos, frequentemente cobrados em questões conceituais.

Tipos de Gráficos Essenciais

Colunas/Barras: Compara valores entre categorias. Exemplo: Vendas por região.
Linhas: Mostra tendências ao longo do tempo. Exemplo: Crescimento de vendas anuais.
Pizza: Exibe proporções de um total. Exemplo: Percentual de despesas por categoria.
Área: Similar ao gráfico de linhas, mas destaca a magnitude acumulada.

Dica de Prova: Escolha o gráfico correto para o tipo de dado. Use colunas para comparações categóricas e pizza para proporções limitadas (máximo 5-7 categorias).

Tabela e Gráfico Dinâmicos

Tabelas Dinâmicas resumem dados interativamente, permitindo agrupamentos e cálculos. Gráficos Dinâmicos visualizam esses resumos e atualizam automaticamente.

Dica de Mestre: Sempre use uma tabela estruturada como fonte para Tabelas Dinâmicas. Isso garante que novos dados sejam incluídos automaticamente.

Exemplo Prático: Crie uma Tabela Dinâmica para resumir vendas por vendedor e trimestre, e adicione um Gráfico Dinâmico de colunas para visualização.

Dicas e Pegadinhas de Prova

Evite erros comuns com estas dicas focadas nas armadilhas mais frequentes em concursos.

1. Operador `&` vs. CONCATENAR

Ambos unem textos, mas o operador `&` é mais curto. Exemplo: `=A1&" "&B1` é igual a `=CONCATENAR(A1; " "; B1)`.

2. Datas são Números

Datas são armazenadas como números sequenciais (ex: 01/01/2024 = 45292). Isso permite cálculos como `=A1+7` para adicionar uma semana.

3. `CONT.SE` vs. `CONT.SES`

`CONT.SE` usa um critério; `CONT.SES` usa múltiplos. Exemplo: `=CONT.SES(A1:A100; "Vendas"; B1:B100; ">1000")`.

4. Apagar vs. Limpar

`Delete` remove conteúdo, mas mantém formatação. Use "Limpar Tudo" para remover conteúdo e formatação.

5. `PROCV` e o `FALSO`

Use `FALSO` para buscas exatas. Exemplo: `=PROCV(A1; B1:D100; 3; FALSO)` evita resultados aproximados errados.

6. `SOMA` Ignora Texto

`=SOMA(A1:A5)` ignora células com texto, somando apenas números. Não gera erro, mas pode enganar em questões.

7. `MÉDIA` Ignora Texto

Calcula a média apenas de números, ignorando texto. Exemplo: `=MÉDIA(10;20;"texto";30)` resulta em 20.

8. Referência Circular

Evite fórmulas que referenciam sua própria célula (ex: `=A1+1` em A1). Isso causa um erro de referência circular.

9. Ocultar vs. Excluir

Ocultar linhas/colunas mantém os dados em cálculos. Excluir remove os dados permanentemente.

10. Precedência de Operadores

Ordem: Parênteses, Exponenciação, Multiplicação/Divisão, Adição/Subtração. Exemplo: `=2+3*5` resulta em 17, não 25.

Exercícios Resolvidos de Concursos

Pratique com questões reais de concursos, acompanhadas de explicações detalhadas para reforçar seu aprendizado.

(CEBRASPE/CESPE - 2023)

Considere a planilha a seguir no Microsoft Excel. Se na célula C2 for inserida a fórmula `=SE(B2>3; "Caro"; "Barato")`, qual será o resultado exibido?

A B
1 Produto Valor
2 Caneta 5
3 Lápis 3

Resposta Correta: "Caro"

Comentário: A função `SE` avalia `B2>3`. Como B2 contém 5, a condição é verdadeira, retornando "Caro". Teste: Em C3, a fórmula retornaria "Barato" (B3=3).

(FCC - 2022)

Em uma planilha do LibreOffice Calc, para fixar a referência à célula A1 ao copiar a fórmula, deve-se usar:

Resposta Correta: `$A$1`

Comentário: O `$` trava a coluna (A) e a linha (1), tornando a referência absoluta. Sem o `$`, a referência mudaria ao copiar (ex: de A1 para B2).

(FGV - 2023)

Dada a fórmula `=CONT.SE(A1:A5; ">10")` aplicada a um intervalo com os valores 5, 10, 15, 20, 25. Qual será o resultado?

Resposta Correta: 3

Comentário: `CONT.SE` conta valores estritamente maiores que 10. No intervalo (5, 10, 15, 20, 25), apenas 15, 20 e 25 atendem, resultando em 3.

(VUNESP - 2022)

A fórmula `=MÉDIA(A1:A4)` é inserida em uma planilha onde A1=10, A2=20, A3="texto", A4=30. Qual o resultado?

Resposta Correta: 20

Comentário: A função `MÉDIA` ignora texto. Soma: 10+20+30=60. Divisão por 3 números: 60/3=20.

(AOCP - 2023)

Qual a finalidade principal do recurso Tabela Dinâmica no Microsoft Excel?

Resposta Correta: Resumir, analisar e apresentar grandes volumes de dados de forma interativa.

Comentário: Tabelas Dinâmicas são usadas para agrupar e resumir dados, como somar vendas por região ou contar registros por categoria, sem precisar de fórmulas complexas.

(IBFC - 2022)

Na célula C1, a fórmula é `=$A1+B$1`. Se esta fórmula for copiada para a célula D2, qual será a fórmula resultante?

Resposta Correta: `=$A2+C$1`

Comentário: `$A` trava a coluna A; a linha 1 torna-se 2 (muda uma linha). B é relativa, vira C (muda uma coluna). `$1` trava a linha 1.

(CONSULPLAN - 2023)

Para somar os valores da coluna C apenas quando a coluna A for "Sul" e a coluna B for "Online", qual função deve ser utilizada?

Resposta Correta: `SOMASES`

Comentário: `SOMASES` é usada para múltiplos critérios. Exemplo: `=SOMASES(C1:C100; A1:A100; "Sul"; B1:B100; "Online")`.

(CEBRASPE/CESPE - 2022)

No LibreOffice Calc, o atalho de teclado para aplicar o formato de Negrito a uma célula selecionada é:

Resposta Correta: `Ctrl + B`

Comentário: No Calc, `Ctrl + B` aplica negrito, enquanto no Excel é `Ctrl + N`. Memorize essas diferenças!

(FGV - 2023)

Dada a fórmula `=ÍNDICE(B1:B10; CORRESP("ItemX"; A1:A10; 0))`, qual valor será retornado?

Resposta Correta: O valor na coluna B correspondente ao "ItemX" na coluna A.

Comentário: `CORRESP` encontra a posição de "ItemX" em A1:A10 (ex: linha 3). `ÍNDICE` retorna o valor de B3. Use 0 para correspondência exata.

(VUNESP - 2023)

Qual o resultado de `=SEERRO(A1/B1; "Erro")` se A1=10 e B1=0?

Resposta Correta: "Erro"

Comentário: Divisão por zero causa um erro (#DIV/0!). `SEERRO` captura isso e retorna "Erro".

Glossário Técnico Simplificado

Conheça os termos mais comuns em planilhas para entender questões e enunciados com precisão.

Argumento

Valores ou referências usados por uma função. Exemplo: Em `=SOMA(A1:A10)`, A1:A10 é o argumento.

Concatenar

Unir textos. Exemplo: `=CONCATENAR("Olá"; " Mundo")` resulta em "Olá Mundo".

Dashboard

Painel visual com gráficos e tabelas para resumir dados. Exemplo: Um dashboard de vendas pode incluir gráficos de pizza e colunas.

Intervalo (Range)

Conjunto de células. Exemplo: `A1:B10` abrange 20 células.

Macro

Sequência de comandos gravada para automatizar tarefas. Exemplo: Uma macro pode formatar uma tabela automaticamente.

Sintaxe

Estrutura correta de uma fórmula ou função. Exemplo: `=SE(A1>10; "Sim"; "Não")`.

Referência

Endereço de uma célula ou intervalo usado em fórmulas. Exemplo: `A1`, `$A$1`.

Tabela Estruturada

Intervalo de dados formatado como tabela, com filtros e expansão automática. Exemplo: Criada com `Ctrl + T` no Excel.
